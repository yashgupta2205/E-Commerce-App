# KLiq – E-Commerce Platform

KLiq is a full-stack e-commerce application consisting of a customer-facing frontend, an admin dashboard, and a backend API for handling authentication, products, carts, and orders.

---

## 🚀 Live Deployments

**Frontend:** https://kliq-frontend-iota.vercel.app/collection  
**Admin Dashboard:** https://kliq-admin.vercel.app/add

---

## 📌 Key Features

### Frontend
- Browse collections and product details  
- Cart management  
- User authentication  
- Checkout and order placement  

### Admin Panel
- Add/Edit/Delete products  
- View and manage orders  
- Inventory management  

### Backend
- JWT-based authentication  
- Product, cart, and order APIs  
- MongoDB database  
- Cloudinary image uploads  

---

## 🏗️ Tech Stack

**Frontend/Admin:** React, React Router, Axios, Tailwind/CSS  
**Backend:** Node.js, Express, MongoDB, Mongoose, Cloudinary, JWT
